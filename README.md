🥗 SPONUT — Suivi Nutritionnel
Application web de suivi nutritionnel journalier, permettant de suivre ses apports en calories, protéines, glucides et lipides avec des conseils 
d'entraînement pour débutants.

📋 Présentation
SPONUT est un projet scolaire de développement web visant à créer une interface de suivi nutritionnel responsive, pensée pour mobile. L'utilisateur peut visualiser ses macronutriments du jour, suivre sa progression hebdomadaire acquérir des connaissances de base pour s'entraîner et naviguer entre les sections de l'application.

✨ Fonctionnalités

Affichage des macronutriments journaliers (Calories, Protéines, Glucides, Lipides)
Barre de progression pour chaque macro
Suivi hebdomadaire avec indicateur du jour actif
Conseils nutritions et entraînements
Menu de navigation via les trois points (⋯)
Header sticky au scroll
Design responsive mobile-first
