# 🥗 SPONUT — Suivi Nutritionnel
 
Application web de suivi nutritionnel journalier, permettant de suivre ses apports en calories, protéines, glucides et lipides avec des conseils d'entraînement pour débutants.
 
---
 
## 📋 Présentation
 
SPONUT est un projet visant à créer une interface de suivi nutritionnel responsive, pensée pour mobile. L'utilisateur peut visualiser ses macronutriments du jour, suivre sa progression hebdomadaire, acquérir des connaissances de base pour s'entraîner et naviguer entre les sections de l'application.
 
---
 
## ✨ Fonctionnalités
 
- Affichage des macronutriments journaliers (Calories, Protéines, Glucides, Lipides)
- Barre de progression pour chaque macro
- Suivi hebdomadaire avec indicateur du jour actif
- Conseils nutrition et entraînements
- Menu de navigation via les trois points (⋯)
- Header sticky au scroll
- Design responsive mobile-first
 
---
 
## 🛠️ Technologies utilisées
 
- **HTML5** pour la structure de la page
- **CSS3** pour la mise en forme, les animations & glassmorphism
- **JavaScript** pour les interactions (menu, navigation)
- **Node.js** pour le backend *(à venir)*
